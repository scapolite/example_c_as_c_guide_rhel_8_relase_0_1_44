## /rule

Implement Blank Screensaver

## /rationale

Setting
the screensaver mode to blank-only conceals the contents of the display
from passersby.

## /description

To
set the screensaver mode in the GNOME3 desktop to a blank screen, add or
set `picture-uri` to `string ''` in
`/etc/dconf/db/local.d/00-security-settings`. For example:

``` 
[org/gnome/desktop/screensaver]
picture-uri=''
```

Once the settings have been added, add a lock to
`/etc/dconf/db/local.d/locks/00-security-settings-lock` to prevent user
modification. For example:

``` 
/org/gnome/desktop/screensaver/picture-uri
```

After the settings have been set, run `dconf update`.

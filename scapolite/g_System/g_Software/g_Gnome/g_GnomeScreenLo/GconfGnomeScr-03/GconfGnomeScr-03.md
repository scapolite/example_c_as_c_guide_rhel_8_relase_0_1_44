## /rule

GNOME Desktop Screensaver Mandatory Use

## /rationale

Enabling
idle activation of the screensaver ensures the screensaver will be
activated after the idle delay. Applications requiring continuous,
real-time screen display (such as network management products) require
the login session does not have administrator rights and the display
station is located in a controlled-access area.

## /description

Run
the following command to activate the screensaver in the GNOME desktop
after a period of inactivity:

``` 
$ sudo gconftool-2 --direct \
  --config-source xml:readwrite:/etc/gconf/gconf.xml.mandatory \
  --type bool \
  --set /apps/gnome-screensaver/idle_activation_enabled true
```

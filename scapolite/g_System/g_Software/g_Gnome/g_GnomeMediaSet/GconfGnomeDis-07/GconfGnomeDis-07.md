## /rule

Disable GNOME Automounting

## /rationale

Disabling
automatic mounting in GNOME can prevent the introduction of malware via
removable media. It will, however, also prevent desktop users from
legitimate use of removable media.

## /description

The
system\'s default desktop environment, GNOME, will mount devices and
removable media (such as DVDs, CDs and USB flash drives) whenever they
are inserted into the system. Disable automount and autorun within GNOME
by running the following:

``` 
$ sudo gconftool-2 --direct \
  --config-source xml:readwrite:/etc/gconf/gconf.xml.mandatory \
  --type bool \
  --set /apps/nautilus/preferences/media_automount false
$ sudo gconftool-2 --direct \
  --config-source xml:readwrite:/etc/gconf/gconf.xml.mandatory \
  --type bool \
  --set /apps/nautilus/preferences/media_autorun_never true
```

## /rule

Verify the UEFI Boot Loader grub.cfg Permissions

## /rationale

Proper
permissions ensure that only the root user can modify important boot
parameters.

## /description

File
permissions for `/boot/efi/EFI/redhat/grub.cfg` should be set to 700. To
properly set the permissions of `/boot/efi/EFI/redhat/grub.cfg`, run the
command:

``` 
$ sudo chmod 700 /boot/efi/EFI/redhat/grub.cfg
```

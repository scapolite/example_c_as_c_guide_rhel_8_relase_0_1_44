## /rule

Verify /boot/grub2/grub.cfg User Ownership

## /rationale

Only
root should be able to modify important boot parameters.

## /description

The
file `/boot/grub2/grub.cfg` should be owned by the `root` user to
prevent destruction or modification of the file. To properly set the
owner of `/boot/grub2/grub.cfg`, run the command:

``` 
$ sudo chown root /boot/grub2/grub.cfg 
```

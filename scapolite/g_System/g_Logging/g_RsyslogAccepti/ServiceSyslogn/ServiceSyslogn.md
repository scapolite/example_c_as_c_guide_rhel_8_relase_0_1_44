## /rule

Enable syslog-ng Service

## /rationale

The
`syslog-ng` service must be running in order to provide logging
services, which are essential to system administration.

## /description

The
`syslog-ng` service (in replacement of rsyslog) provides syslog-style
logging by default on Debian 8. The `syslog-ng` service can be enabled
with the following command:

``` 
$ sudo systemctl enable syslog-ng.service
```

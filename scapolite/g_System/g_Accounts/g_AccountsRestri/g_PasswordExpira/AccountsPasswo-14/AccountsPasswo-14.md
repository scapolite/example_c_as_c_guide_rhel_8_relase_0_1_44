## /rule

Set Password Warning Age

## /rationale

Setting
the password warning age enables users to make the change at a practical
time.

## /description

To
specify how many days prior to password expiration that a warning will
be issued to users, edit the file `/etc/login.defs` and add or correct
the following line:

``` {html="http://www.w3.org/1999/xhtml" ns0="http://checklists.nist.gov/xccdf/1.1"}
PASS_WARN_AGE 
```

The DoD requirement is 7. The profile requirement is ``.

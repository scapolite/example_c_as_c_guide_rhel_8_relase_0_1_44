## /rule

Disable Kernel Support for USB via Bootloader Configuration

## /rationale

Disabling
the USB subsystem within the Linux kernel at system boot will protect
against potentially malicious USB devices, although it is only practical
in specialized systems.

## /description

All
USB support can be disabled by adding the `nousb` argument to the
kernel\'s boot loader configuration. To do so, append \"nousb\" to the
kernel line in `/etc/default/grub` as shown:

``` 
kernel /vmlinuz-VERSION ro vga=ext root=/dev/VolGroup00/LogVol00 rhgb quiet nousb
```

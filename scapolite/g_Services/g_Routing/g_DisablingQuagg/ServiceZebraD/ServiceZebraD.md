## /rule

Disable Quagga Service

## /rationale

Routing
protocol daemons are typically used on routers to exchange network
topology information with other routers. If routing daemons are used
when not required, system network information may be unnecessarily
transmitted across the network.

## /description

The `zebra` service can be disabled with the following command:

``` 
$ sudo systemctl disable zebra.service
```

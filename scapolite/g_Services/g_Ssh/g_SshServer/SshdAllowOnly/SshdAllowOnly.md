## /rule

Allow Only SSH Protocol 2

## /rationale

SSH
protocol version 1 is an insecure implementation of the SSH protocol and
has many well-known vulnerability exploits. Exploits of the SSH daemon
could provide immediate root access to the system.

## /description

Only
SSH protocol version 2 connections should be permitted. The default
setting in `/etc/ssh/sshd_config` is correct, and can be verified by
ensuring that the following line appears:

``` 
Protocol 2
```

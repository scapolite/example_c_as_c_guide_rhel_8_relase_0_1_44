## /rule

Set SSH Client Alive Max Count

## /rationale

This
ensures a user login will be terminated as soon as the
`ClientAliveInterval` is reached.

## /description

To
ensure the SSH idle timeout occurs precisely when the
`ClientAliveInterval` is set, edit `/etc/ssh/sshd_config` as follows:

``` {html="http://www.w3.org/1999/xhtml" ns0="http://checklists.nist.gov/xccdf/1.1"}
ClientAliveCountMax 
```

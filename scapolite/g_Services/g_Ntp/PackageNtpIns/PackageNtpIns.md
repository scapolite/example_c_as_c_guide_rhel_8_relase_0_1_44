## /rule

Install the ntp service

## /rationale

Time
synchronization (using NTP) is required by almost all network and
administrative tasks (syslog, cryptographic based services
(authentication, etc.), etc.). Ntpd is regulary maintained and updated,
supporting security features such as RFC 5906.

## /description

The
ntpd service should be installed.

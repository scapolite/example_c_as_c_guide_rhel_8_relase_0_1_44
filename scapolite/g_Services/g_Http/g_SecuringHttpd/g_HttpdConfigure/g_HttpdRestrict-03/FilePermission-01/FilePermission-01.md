## /rule

Set Permissions on All Configuration Files Inside /etc/httpd/conf.d/

## /rationale

Access
to the web server\'s configuration files may allow an unauthorized user
or attacker to access information about the web server or to alter the
server\'s configuration files.

## /description

To properly set the permissions of `/etc/http/conf.d/*`, run the
command:

``` 
$ sudo chmod 0640 /etc/http/conf.d/*
```
